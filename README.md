amazon-prime-video-powerbi-dashboard

📌 Project Overview
This project is a data analysis dashboard created using Power BI to explore the Amazon Prime Video dataset. The dashboard provides insights about movies and TV shows available on the platform, including genre distribution, country-wise content, and release trends.

 🎯 Objective
The main objective of this project is to analyze Amazon Prime Video content and present meaningful insights using interactive data visualizations.

 📊 Dashboard Insights
- Total number of Movies and TV Shows available on Amazon Prime Video
- Distribution of content by country
- Genre-wise analysis of movies and TV shows
- Year-wise content release trends
- Ratings and category analysis

🛠 Tools Used
- Power BI
- Excel / CSV dataset
- Data Visualization techniques

 📂 Files in this Repository
- `Amazon Prime video.pbix` – Power BI dashboard file
- `amazon_prime_titles.csv` – Dataset used for analysis
